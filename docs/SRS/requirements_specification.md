# Спецификация требований к программному обеспечению
# TaxForex

**Версия:** 1.0
**Дата:** 15.03.2024

## 1. Введение

### 1.1. Назначение
Документ описывает требования к веб-приложению TaxForex для расчета налогообложения операций в иностранной валюте.

### 1.2. Область действия
Приложение предназначено для использования индивидуальными предпринимателями в Российской Федерации.

### 1.3. Определения
- **ИП** - Индивидуальный предприниматель
- **ЦБ РФ** - Центральный банк Российской Федерации
- **КУДиР** - Книга учета доходов и расходов

## 2. Общее описание

### 2.1. Функции продукта
- Запрос и отображение курсов валют ЦБ РФ
- Ввод и хранение операций в иностранной валюте
- Автоматический расчет рублевого эквивалента
- Формирование налоговых отчетов
- Экспорт данных

### 2.2. Классы пользователей
- **ИП** - основной пользователь, ведет учет валютных операций
- **Бухгалтер** - может использовать для подготовки отчетности

## 3. Функциональные требования

### FR1: Управление операциями
**Приоритет:** Высокий
**Описание:** Система должна позволять пользователю:
- Добавлять новые операции (доход/расход)
- Указывать валюту, сумму, дату операции
- Автоматически определять курс ЦБ РФ на дату операции
- Просматривать историю операций
- Редактировать и удалять существующие операции

### FR2: Формирование отчетов
**Приоритет:** Высокий
**Описание:** Система должна формировать отчеты:
- Сводный отчет по валютам за период
- Итоговые суммы в рублях для налоговой декларации
- Экспорт в PDF и Excel formats

## 4. Нефункциональные требования

### NF1: Производительность
- Время отклика интерфейса: < 100 мс
- Время загрузки курсов валют: < 2 секунд

### NF2: Надежность
- Доступность: 99.5%
- Резервное копирование данных локально

### NF3: Удобство использования
- Интерфейс должен быть интуитивно понятен для неподготовленных пользователей
- Добавление операции должно занимать не более 3 действий

## 5. Интерфейсы

### 5.1. Пользовательский интерфейс
- Веб-интерфейс с адаптивным дизайном
- Поддержка современных браузеров

### 5.2. Внешние интерфейсы
- API ЦБ РФ для получения курсов валют
- Локальное хранилище браузера
